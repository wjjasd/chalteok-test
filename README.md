# 찰떡 (Chalteok)

관계 적합도를 구조화된 설문으로 정량 시각화하는 성찰 보조 도구.

- **URL**: https://chalteok.kro.kr
- **상태**: 구현 착수 전 (기획서 v1.0 완료)
- **작업 기록·의사결정**: brain vault `10_projects/chalteok/` 참조

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| 프레임워크 | Next.js (App Router) |
| 언어 | TypeScript |
| 스타일링 | Tailwind CSS v4 |
| 상태 관리 | Zustand v5 |
| 차트 | Recharts |
| 배포 | Vercel |

---

## 로컬 실행

```bash
npm install
npm run dev
```

`http://localhost:3000` 에서 확인
